Tailwind CSS Lookup Settings

When disabled, the extension won't inject any scripts into pages.

About

Tailwind CSS Lookup lets you inspect any element's Tailwind classes and look up what CSS they generate.

  • Hover over elements to see their Tailwind classes
  • Search for any Tailwind class to see its CSS
  • Click to copy classes to clipboard
  • See a summary of all Tailwind classes used on a page

Keyboard Shortcuts

  • Esc — Exit inspect mode
  • Click — Copy element's Tailwind classes
