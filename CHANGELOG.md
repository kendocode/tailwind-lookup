# Changelog

All notable changes to Tailwind CSS Lookup will be documented in this file.

## [0.1.0] - 2026-02-24

### Added
- Inspect mode: hover over elements to see their Tailwind classes with CSS output
- Class search: look up any Tailwind utility class in the popup
- Click to copy: click an element in inspect mode to copy its classes
- Page summary: count of unique Tailwind classes on the current page
- Tailwind CSS v3 class database
- Light/dark theme support
- Options page for preferences
