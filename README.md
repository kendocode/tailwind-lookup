# Tailwind CSS Lookup

Inspect Tailwind CSS classes on any element in your browser. Look up any class to see the CSS it generates. The missing browser companion for Tailwind developers.

## Features

- **Inspect Mode** - Hover over any element to see its Tailwind classes with the CSS they generate
- **Class Search** - Type any Tailwind class in the popup to see its CSS output
- **Click to Copy** - Click an element in inspect mode to copy its Tailwind classes
- **Page Summary** - See how many unique Tailwind classes are used on the current page

## Installation

### From Chrome Web Store (coming soon)
1. Install from the Chrome Web Store
2. Click the extension icon to open the popup
3. Use "Inspect Element" to hover over elements, or search for classes directly

### Local Development
1. Open `chrome://extensions/`
2. Enable "Developer mode"
3. Click "Load unpacked" and select this directory
4. Visit any page using Tailwind CSS (e.g., tailwindcss.com)
