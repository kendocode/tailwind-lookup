# Privacy Policy for Tailwind CSS Lookup

**Last updated:** 2026-02-24

## Overview

Tailwind CSS Lookup is committed to protecting your privacy.

## Data Collection

Tailwind CSS Lookup does **not** collect, transmit, or share any personal data.

### What the extension accesses:
- **Page DOM** (when inspect mode is active): Reads CSS class names from page elements to identify Tailwind classes. All processing is local in your browser.
- **Extension settings**: Stored locally using Chrome's storage API.

### What the extension does NOT do:
- Does not collect analytics or usage data
- Does not use cookies or tracking pixels
- Does not transmit any data to external servers
- Does not access your browsing history
- Does not modify page content (read-only inspection)

## Data Storage

All data is stored locally on your device. No data is stored on external servers.

## Third-Party Services

This extension does not integrate with or send data to any third-party services.

## Contact

If you have questions about this privacy policy, please file an issue on the extension's support page.
